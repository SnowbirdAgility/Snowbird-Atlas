02  Logo Cleanup & Recommendations

Honest read: strong concept, raster reality.

The current files are AI-generated preview images — opaque white backgrounds, soft drop shadows, and the wordmark baked in as low-resolution pixels. The concept is good; the production art is not there yet. Element by element:

KEEP

The bird / upswept wing

This is the equity of the mark — distinctive, optimistic, and the one element that survives shrinking. Keep it dominant. Flatten its internal gradient to two tones so it holds at small sizes.

KEEP

The orbital ring

Reinforces “atlas / global reach” and adds motion. Keep it — but drop the thin white inner highlight that creates a halo on dark and vanishes on light. One solid stroke.

FIX

The wireframe globe

The most fragile element. Dense meridians plus the network “connection dots” collapse into a gray smudge below ~48px. Reduce to a few meridians and an equator; remove the network dots entirely.

FIX

The wordmark

Currently baked-in raster text at low resolution. Re-typeset as live vector in Sora (done throughout this guide). Keep the letter-spaced “ATLAS” with flanking rules — it’s a nice device.

SIMPLIFY

Below ~32px → simplified mark

Use the geometric simplified mark (reduced globe + clean ring + flat wing). It is the same idea with the noise removed.

SIMPLIFY

Below ~24px → app icon

Drop the globe and ring; a navy rounded square with the cyan wing reads cleanly as a favicon, app tile, or social avatar.

04  Color

Navy anchors. Cyan signals. Gray supports.

A disciplined five-color palette. Midnight Navy carries the brand; Electric Cyan is the single accent used sparingly for emphasis and energy; grays and Cloud White do the structural work.

Midnight Navy
#0B1F3A · primary
Primary brand color. Backgrounds, headlines, the wordmark, large fields. The default ink.
Slate Blue
#1E3A5F · secondary
Secondary surfaces, gradients with navy, hover states, the orbital ring.
Electric Cyan
#22D3EE · accent
Accent only. Ticks, links, key CTAs, data highlights. Never as body text or large fields.
Cloud White
#F9FAFB · surface
Default light background and reversed text on navy. Soft, never pure #FFF for large fields.
Cool Gray
#9CA3AF · support
Hairlines, captions, the globe wireframe, disabled states, secondary metadata.
